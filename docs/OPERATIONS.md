# Operations / 运维

## English
### Local dev
```bash
cd /path/to/codex-remote-bridge
npm install
npm run setup
npm run dev:relay
npm run build:desktop
npm run start:desktop
```

### Checks
```bash
npm run typecheck
npm run build
```

### Packaging
```bash
npm run dist:desktop:zip
npm run dist:desktop
```
